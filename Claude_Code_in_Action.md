# Claude Code in Action - Quick Notes

## What is Claude Code?

* AI coding assistant that works with your codebase.
* Can read, understand, modify, and create code.
* Helps automate development tasks.

---

## Key Concepts

### 1. Context is King 👑

* Give Claude access to the right files and project information.
* Better context = Better code suggestions.

---

### 2. Think → Plan → Execute

For complex tasks:

1. Analyze
2. Create a plan
3. Implement changes

---

### 3. Use Custom Commands

Save repetitive prompts and workflows.

**Examples:**

* Generate Jira stories
* Create unit tests
* Perform code reviews
* Generate documentation

---

### 4. Control Context

* Include only relevant files.
* Too much context can reduce effectiveness.

---

### 5. MCP Servers = Superpowers

Connect Claude to external tools and services.

**Examples:**

* Browser automation
* External APIs
* Specialized development workflows

---

### 6. GitHub Integration

Use Claude for:

* Pull Request reviews
* Code analysis
* Workflow automation
* Development collaboration

---

### 7. Hooks = Automation Triggers

Run actions automatically before or after tasks.

**Examples:**

* Validation checks
* Test execution
* Code formatting
* Security checks

---

### 8. Claude Code SDK

Build custom integrations and automations using Claude.

---

## Practical Use Cases

### Development

* Write code
* Refactor code
* Generate tests
* Debug issues

### DevOps / SRE

* Generate automation scripts
* Create Terraform/YAML
* Analyze logs
* Draft RCA documents
* Create runbooks

### Project Management

* Generate Jira stories
* Create documentation
* Summarize changes

---

## Golden Rules

* Provide clear context
* Let Claude plan before coding
* Review all generated code
* Automate repetitive tasks with commands and hooks
* Use GitHub integration for reviews
* Never share sensitive information

---

# 5 Things to Remember

1. **Context**
2. **Planning**
3. **Custom Commands**
4. **MCP Servers**
5. **GitHub Integration**

---

# One-Line Summary

**Claude Code is an AI pair programmer that helps you understand, plan, write, review, and automate code using context, custom commands, hooks, MCP servers, and GitHub integrations. 🚀**
